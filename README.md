# Club Leaders Portal

Portal for [@hackclub](https://github.com/hackclub/) club leaders to manage their clubs

## To run
~~~ bash
cp example.env .env # and then like fill out the env
npm i
npm run dev
~~~

## Features
- View events (loaded from airtable)
- View and manage your clubs info
- Mark events as completed
- Admin portal + analytics
